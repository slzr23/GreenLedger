Plateforme Carbone Entreprise

Transformez vos actions écologiques en crédits carbone traçables

La plateforme permet aux entreprises de piloter des quotas annuels, de valider des actions terrain via la DAO, puis de compenser l’empreinte carbone avec un jeton dédié.

Accéder Connexion MetaMask sur l’étape suivante
Validation collective des actions
Récompense en jetons carbone
Compensation vérifiable on-chain

Schéma du cycle carbone

1. Employé soumet une preuve Proof
2. Vote DAO (pour / contre) Vote
3. Action validée → mint ECO + ECO
4. Compensation empreinte → burn ECO Burn
